## GitHub骚操作学习

#### 常用词解释

| 名称   | 描述                                 |
| ------ | ------------------------------------ |
| watch  | 会持续收到该项目的动态               |
| fork   | 复制某个项目到自己的GitHub仓库中     |
| star   | 可以理解为点赞                       |
| clone  | 将项目下载至本地                     |
| follow | 关注你感兴趣的作者，会收到他们的动态 |
